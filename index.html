About

My name is Kyle Jang, and I am going into my fourth year in the University of Toronto. I am studying as a Computer Science major, Statistics minor, and Sociology minor.

Ever since I was a child, I was always fascinated with technology. I started to really love computers when I got my first netbook all the way back in Elementary school. I remember wanting to play so many video games, but being held back by the poor netbook hardware specifications. From there, I started learning about computer hardware in order to understand why I couldn't run certain games, and this eventually led me to develop an interest in computer software.

I had my first experiences with programming during university, and I quickly developed a love for computer science after my first year. During my three years of university so far, I learned how to code in many different programming languages such as Java, Python, and C. I also learned about Agile software developement and how to effectively work in teams to build applications. In addition to learning programming languages and building applications, I learned about data structures and code optimization. As I started gaining more experience in programming, my desire for app development grew. As a result, I started learning HTML, CSS, Javascript, and React during my free time so that I could create my own applications from the ground up.

I have completed a few personal projects, which include a workout journal app, a personal notepad app, and a recipe finder app. I completed these projects in order to practice prgramming with React, HTML, CSS, and Javascript. When I am not coding, I really enjoy working out, playing with my dog, and playing PC games. Some of my favourite video games include Monster Hunter World, Red Dead Redemption 2, Divinity Original Sin 2, and Witcher 3.

Projects

Workout Journal:

Since I enjoy working out, I decided to create a workout journal app with my brother which allows users to easily keep track of their workouts and track their progress. This project helped me to learn how to use React Native, and also taught me a lot about mobile app developement.


Personal Notepad:

I created this notepad app which allows users to save and delete notes, and personalize the color of each note. I completed this project in order to practice using HTML and Javascript.



Recipe Finder:

I created this application with my brother which the uses Edamam Recipe Search API to look up 4 different recipes for specified foods. We created this application in order to practice working with APIs and to help us figure out what to cook with the many different ingredients that are in our house.
