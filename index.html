About

My name is Kyle Jang, and I am a third year student in the University of Toronto. I am studying as a Computer Science major, Statistics minor, and Sociology minor.

Ever since I was a child, I was always fascinated with technology. I started to really love computers when I got my first netbook all the way back in Elementary school. I remember wanting to play so many video games, but being held back by the poor netbook specifications. From there, I started to get into learning about computer hardware in order to understand why I couldn't run certain games, and this eventually led me to develop an interest in computer software.

I only really started coding when I started University, but I quickly developed a love for coding and Computer Science after my first year. During my first two years in university, I learned how to code in Java, Python and C. I also learned how to work in a team to build an application and how to optimize code so that it has a faster run-time.

I have a few personal projects that I have completed, which include a chrome extension, a personal notepad app, and a recipe finder app. Since I did not learn much about front end programming from university so far, I completed these projects in order to teach myself how to code in HTML, CSS, and Javascript. During my free time when I am not coding, I really enjoy working out, playing with my dog, and playing PC games. Some of my favourite video games include Overwatch, Rainbow Six Siege, Divinity Original Sin 2, and the Witcher 3.

Projects

Custom Shortcuts Chrome Extension:

I created a Chrome extension that allows me to input my desired website URLs into eight different shortcuts. I made this extension in order to get practice in coding with JavaScript and jQuery, and to also make it easier for me to access my most used websites.


Personal Notepad:

I created this notepad app which allows users to save and delete notes, and personalize the color of each note. I completed this project in order to practice using HTML and Javascript.



Recipe Finder:

I created this application with my brother which the uses Edamam Recipe Search API to look up 4 different recipes for specified foods. We created this application in order to practice working with APIs and to help us figure out what to cook with different ingredients in our house.
